# NoteGraph

Десктоп-приложение для управления взаимосвязанными текстовыми заметками.

## Описание

NoteGraph - это система управления персональной базой знаний, позволяющая создавать текстовые заметки с поддержкой Markdown и устанавливать между ними семантические связи по принципу вики-системы. Приложение превращает коллекцию разрозненных заметок в единую сеть знаний с возможностью визуализации в виде интерактивного графа.

## Основные возможности

- ✅ Создание и редактирование заметок с поддержкой Markdown
- ✅ Установление двунаправленных связей между заметками через синтаксис `[[название]]`
- ✅ Полнотекстовый поиск по содержимому заметок
- ✅ Визуализация структуры знаний в виде интерактивного графа
- ✅ Локальное хранение данных в базе данных SQLite
- ✅ Автосохранение изменений
- ✅ Кроссплатформенность (Windows, macOS, Linux)

## Системные требования

- Java Runtime Environment (JRE) 17 или выше
- Операционная система: Windows 10+, macOS 10.14+, или Linux
- Минимум 2 ГБ оперативной памяти
- Минимум 200 МБ свободного дискового пространства

## Установка и запуск

### Сборка проекта

```bash
# Клонирование репозитория
git clone https://github.com/T2F2R/notegraph
cd notegraph

# Компиляция проекта
mvn clean compile

# Запуск приложения
mvn javafx:run

# Сборка JAR файла
mvn clean package
```

### Запуск из командной строки

```bash
java --module-path /path/to/javafx-sdk/lib --add-modules javafx.controls,javafx.fxml,javafx.web -jar target/notegraph-1.0-SNAPSHOT.jar
```
