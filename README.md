# 🧠 NoteGraph

**Десктоп-приложение для работы с взаимосвязанными текстовыми заметками**

Локальная база знаний с поддержкой Markdown, тегов, двунаправленных связей между заметками и интерактивной визуализацией графа.

## 📖 О проекте

**NoteGraph** — это инструмент для создания персональной базы знаний по принципу *Zettelkasten*: каждая заметка — отдельный файл `.md`, заметки связываются между собой через `[[wikilink]]`, а структура знаний визуализируется в виде интерактивного графа.

Все данные хранятся **локально** в открытом формате Markdown — без облака, без подписок, без привязки к конкретному приложению.

## ✨ Возможности

### 📝 Заметки
- Создание, редактирование и удаление заметок в формате Markdown
- Организация заметок в иерархическую структуру папок
- Ежедневные заметки (Daily Notes) с автоматическим названием по дате
- Режимы редактирования и предварительного просмотра
- Автоматическое сохранение

### 🔗 Связи и теги
- Двунаправленные связи между заметками: `[[Название заметки]]`
- Ссылки с произвольным текстом: `[[Заметка|отображаемый текст]]`
- Автосоздание заметки при переходе по несуществующей ссылке
- Теги `#тег` — кликабельные, с быстрым поиском всех заметок по тегу
- Автоматическая индексация связей и тегов при сохранении

### 🕸️ Визуализация графа
- Интерактивный граф связей между заметками
- Force-directed размещение узлов (физическая симуляция)
- Перемещение, масштабирование и панорамирование
- Переход к заметке кликом по узлу

### 🔍 Поиск и навигация
- Полнотекстовый поиск по содержимому заметок
- Поиск по названиям и по тегам
- Закладки для быстрого доступа
- Список последних открытых заметок

### 🎨 Персонализация
- Светлая и тёмная темы оформления
- Настройка шрифта и его размера
- Локализация интерфейса: 🇷🇺 русский / 🇬🇧 английский

### 🔒 Безопасность
- Изоляция данных в выделенном каталоге хранилища (vault)
- Защита от path traversal атак
- Экранирование HTML и защита от XSS при рендеринге
- Соответствует требованиям ГОСТ Р ИСО/МЭК 15408-3-2013 (ОУД2) и Приказа ФСТЭК №76 (6 УД)

## 🚀 Быстрый старт

### Требования

- **Java 17** или выше (JDK)
- **Apache Maven 3.8** или выше

### Сборка из исходников

```bash
git clone https://github.com/T2F2R/notegraph
cd notegraph
mvn clean package
```

Готовый JAR появится в каталоге `target/`.

### Запуск

```bash
java -jar target/notegraph-1.0.0.jar
```

При первом запуске приложение создаст каталог хранилища (`vault`) и служебную папку `.notegraph` с метаданными.

## 📁 Структура хранилища

```
vault/
├── .notegraph/             # служебные данные (не редактировать вручную)
│   ├── metadata.json        # настройки, закладки, последние заметки
│   ├── links_index.json      # индекс связей между заметками
│   └── tags.json            # индекс тегов
├── Папка/
│   ├── Заметка 1.md
│   └── Заметка 2.md
└── Заметка 3.md
```

Пример заметки:

```markdown
# Заголовок заметки

Обычный текст в формате **Markdown**.

Связь с другой заметкой: [[Другая заметка]]
Связь с алиасом: [[Другая заметка|кликни сюда]]

#тег #ИИ
```

Поскольку заметки — это обычные `.md` файлы, их можно открывать и редактировать в любом текстовом редакторе.

## 🧪 Тестирование

Проект покрыт автоматизированными тестами:

- **JUnit 5** — модульные тесты бизнес-логики
- **TestFX** — UI-тесты, имитирующие действия пользователя
- **Awaitility** — проверка асинхронных операций

Тестовые сценарии включают создание/удаление заметок и папок, работу со связями, навигацию по дереву и автоматическое создание заметок при переходе по wikilink.
